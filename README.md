*30 Days C Programming Challenge
 About This Challenge
I started this 30 Days C Programming Challenge to improve my programming skills, strengthen problem-solving, and build consistency in coding.
This challenge covers:
Basics of C Programming
Input/Output
Conditional Statements
Loops
Arrays
Strings
Functions
Real-life mini programs
My goal is to practice coding every day and upload my progress to GitHub.
